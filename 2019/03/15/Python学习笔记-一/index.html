Python学习笔记(一)

发表于 2019-03-15

前言

    虽然说前面有过一些python的基础，不过也只是达到了入门的级别，也只仅仅是知道了基础的数据类型以及语法，更为高深的用法却还没有摸透，况且Python在当前情景下发展也极为迅速，作为选择IT行业的我来说，也需要去积极的学习新的知识，保持自己的学习能力也是一项重要的技能。

Python的特点

    根据我近期来的学习，我印象最深刻的就是函数式编程的思想。按照我的理解，函数式编程，就是把函数名当成对象来使用的一种思想，通过这种思想，衍生出了各种用法，比如装饰器，就是一个很典型的例子。接下来做一些使用的笔记。

笔记

    先来一些简单的例子

函数作为参数

map()映射函数

    这段代码很好的体现了map函数的用法，前一个参数是函数，后一个参数是一个可迭代的序列，比如列表，将序列中的每一个元素都作为f的参数，形成了新的序列，比如开始的序列是[1,2,3]，如上使用map()后，序列就为[f(1),f(2),f(3)].

reduce()映射函数2

    reduce()函数所需的参数与map相似，不过他输出的是函数反复调用后的值，而且接受的函数f()必须带两个参数，上述代码的功能就是，计算出list中元素的乘积，比如list=[1,2,3]，经过reduce()后就得到a=123，顺带一提，reduce可以在后面添加计算的初始值。

filter()筛选元素

    filter()所需的参数也和map()相似，不过它的作用在于判断，判断元素带入函数后的值，True留下，False舍弃，留下的构成一个新列表。
